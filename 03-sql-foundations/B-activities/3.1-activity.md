# Activity 3.1: SQL Fundamentals Self-Check (10-15 min)

## Instructions

Answer the questions below without checking notes first. Then review your answers.

---

## Questions

1. What does SQL stand for?
2. Is SQL declarative or imperative? Explain in one sentence.
3. What is ANSI SQL?
4. Name three SQL dialects.
5. Roughly what percentage of SQL is the same across most databases?
6. True or False: If you learn PostgreSQL SQL, you cannot use MySQL later.
7. Give one reason PostgreSQL is used in this course.

---

## Suggested Answers

1. Structured Query Language.
2. Declarative: you state the result you want, and the database decides how to execute it.
3. The international SQL standard that defines core syntax supported across database systems.
4. Examples: PostgreSQL SQL, MySQL, T-SQL, PL/SQL, SQLite SQL, Snowflake SQL, BigQuery SQL.
5. Around 95% is shared core SQL.
6. False.
7. Example reasons: industry usage, strong ANSI alignment, free/open source, good docs.
