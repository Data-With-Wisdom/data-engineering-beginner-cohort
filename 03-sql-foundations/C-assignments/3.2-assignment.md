# Assignment 3.2: pgAdmin Setup and First SQL Execution

## Objective
Confirm your local PostgreSQL + pgAdmin environment is ready for the course.

## Tasks
1. Open pgAdmin and connect to your PostgreSQL server.
2. Create a database for this cohort module.
3. Run and save outputs for:
   - SELECT version();
   - SELECT CURRENT_DATE, CURRENT_TIMESTAMP;
   - SELECT 2 + 2 AS result;
4. Create one table, insert at least 2 rows, and query the table.
5. Save your SQL and screenshots in one submission file.

- Include SQL code blocks and screenshots.
