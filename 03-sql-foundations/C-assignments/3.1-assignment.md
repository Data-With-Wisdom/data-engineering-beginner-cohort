# Assignment 3.1: SQL Dialect Comparison

## Objective

Show that you understand standard SQL versus dialect-specific differences.

---

## Scenario

You join a team that currently uses PostgreSQL but may migrate some workloads to SQL Server or BigQuery.

---

## Tasks

1. Write one ANSI SQL query that selects all customers in Lagos.
2. Show how to limit results to 5 rows in:
   - PostgreSQL/MySQL
   - SQL Server
   - Oracle (modern syntax)
3. Show one string concatenation example in:
   - PostgreSQL
   - MySQL
   - SQL Server
4. In 5-8 sentences, explain why learning one SQL dialect makes it easier to learn others.

---

## Submission format

- File name: `3.1-assignment-submission.md`
- Include code blocks for each SQL example.
- Keep explanations concise and clear.
