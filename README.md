# Data Engineering Beginner Cohort

This repository contains the learning materials, activities, and assignments for the Data Engineering Beginner Cohort.

Course content is released progressively each week to match live sessions and ensure everyone moves at the same pace.

## Current Course Outline (Weeks 1-7)

### Pre-Course Setup (00-setup)

Focus:
1. Environment setup (Python, VS Code, Jupyter)
2. Creating accounts (GitHub)
3. Testing installations with a "Hello World" script
4. Welcome pack (expectations and time commitment)
