# Activity 0.2: VS Code Installation, Setup, and Verification

**Goal:** Install Visual Studio Code (VS Code), set it up for Python work, and confirm it runs on your machine.  
**Estimated time:** 15-25 minutes

## Step 1: Download VS Code
- Go to the official VS Code downloads page and choose the installer for your operating system:
  - [VS Code Downloads](https://code.visualstudio.com/Download)

## Step 2: Install VS Code
### Windows
1. Run the installer.
2. Check **"Add to PATH"** when prompted.
3. Complete the installation.

### macOS
1. Open the downloaded `.zip`.
2. Drag **Visual Studio Code** into your **Applications** folder.

### Linux (Debian/Ubuntu)
1. Download the `.deb` package.
2. Install with:

```bash
sudo dpkg -i code_*.deb
```

## Step 3: Basic Setup
1. Open VS Code.
2. Install the **Python** extension:
   - Go to **Extensions** (left sidebar).
   - Search for **Python** (by Microsoft) and click **Install**.
3. (Optional) Sign in to enable Settings Sync.

## Step 4: Verify the Installation
Open a terminal and run:

```bash
code --version
```

If the command is not found:
- Restart your terminal and try again.
- On macOS, open VS Code and run **"Shell Command: Install 'code' command in PATH"** from the Command Palette.

## Step 5: Quick Test in VS Code
1. Create a new file named `hello.py`.
2. Add this code:

```python
print("hello, data engineering!")
```

3. Run it using the **Run Python File** button or by using the terminal:

```bash
python hello.py
```

- Paste a screenshot or the output showing `code --version` and the "hello, data engineering!" message.
